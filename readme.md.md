# Lab 3: K-Means Clustering with Real Sales Data

## Overview

This Jupyter Notebook lab teaches customer segmentation using K-Means clustering. Students work with simulated Amazon sales data to group customers based on purchase behavior.

## Learning Objectives

- Create and explore datasets using pandas
- Apply K-Means clustering with scikit-learn
- Visualize clusters using matplotlib
- Interpret results for business decision-making
- Understand how `k` (number of clusters) affects segmentation
- Perform two-feature clustering with categorical data encoding

## Lab Structure

| Section | Topic |
|---------|-------|
| Steps 1-2 | Import libraries & create dataset |
| Steps 3-4 | Data exploration & preparation |
| Steps 5-10 | K-Means modeling & basic visualizations |
| Steps 11-14 | Professional charts & business reporting |
| Exercise 1 | Change `n_clusters` from 3 to 4 |
| Exercise 2 | Add high-value orders ($700-$1000) |
| Exercise 3 | Two-feature clustering (Amount + Category) |

## Technologies Used

- Python 3
- pandas – data manipulation
- scikit-learn – KMeans clustering, StandardScaler
- matplotlib – data visualization

## Key Concepts

- Unsupervised learning
- Customer segmentation
- Cluster centers and interpretation
- Feature scaling importance
- One-hot vs ordinal encoding

## Results

The lab demonstrates how different `k` values reveal different customer segments, how new data shifts cluster centers, and how combining multiple features creates richer customer profiles for targeted marketing.
